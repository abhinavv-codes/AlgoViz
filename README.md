# 🚀 Algorithm Visualizer

This project is a web-based Algorithm Visualizer built using HTML, CSS, and JavaScript. It helps users understand how different algorithms work through interactive visual representations.

---

## 📌 Features

### 🔢 Sorting Algorithms

* Bubble Sort
* Merge Sort
* Quick Sort
* selection
* insertion 

### 🔍 Searching Algorithms

* Linear Search
* Binary Search

### 🧭 Graph Algorithms would be added in future 

* BFS (Breadth First Search)
* DFS (Depth First Search)

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## ⚙️ How to Run

1. Download or clone the repository
2. Open the folder
3. Double-click on `index.html`

OR

Right-click → Open with browser

---

## 🧠 How It Works

* Each algorithm is implemented using JavaScript
* DOM manipulation is used to animate steps
* Visual feedback is given using colors and transitions
